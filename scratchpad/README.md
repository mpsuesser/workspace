# scratchpad

Throwaway experiments, spikes, and one-off scripts.

This workspace member is intentionally loose: lint rules and conventions
that apply elsewhere in the repo are relaxed or disabled here. Code in
`scratchpad/` is not expected to be production-ready and may be deleted
without notice.

Use it for:

- Quick API explorations against `@workspace/*` packages
- Testing Effect patterns before promoting them into a binding or app
- Reproducing bugs in isolation
