# Developer Tools

Raycast provides several tools to smoothen your experience when building extensions:

* [Manage Extensions Command](developer-tools/manage-extensions-command.md) *- A Raycast command to manage your extensions, add new command, etc.*
* [CLI](developer-tools/cli.md) *- A CLI to build, develop, and lint your extension*
* [ESLint](developer-tools/eslint.md) *- An ESLint configuration helping you follow best practices as you build your extension*
* [Forked Extensions (community tool)](developer-tools/forked-extensions.md) - *The extension for helping you manage your forked Raycast extensions*
* [VS Code (community tool)](developer-tools/vscode.md) *- A VS Code extension to enhance your development experience*
