# Feedback

Raycast has several ways to provide feedback to the user:

* [Toast](feedback/toast.md) *- when an asynchronous operation is happening or when an error is thrown*
* [HUD](feedback/hud.md) *- to confirm an action worked after closing Raycast*
* [Alert](feedback/alert.md) *- to ask for confirmation before taking an action*
