# dprint

dprint is a pluggable code formatter written in Rust.

When working with dprint, read the relevant local reference first. The files in `references/` are a cleaned Markdown mirror of the dprint documentation and plugin configuration schemas.

## Common tasks

- Install and initialize dprint: [install](references/install.md), [setup](references/setup.md)
- Configure `dprint.json` / `dprint.jsonc`: [configuration](references/config.md)
- Use the CLI: [CLI](references/cli.md)
- Set up CI: [continuous integration](references/ci.md)
- Use global config: [global configuration](references/global-config.md)
- Add or configure plugins: [plugins overview](references/plugins.md)
- Create a plugin: [plugin development](references/plugin-dev.md)
